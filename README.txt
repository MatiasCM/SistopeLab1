Laboratorio 1 de Sistemas operativos 1/2025

Para compilar el programa ingresar en teminal el comando

make

Para ejecutar el programa ingresar comando

./desafio1 -t 10 -M 10 -p 4

Comando de ejemplo creara el token(-t) 10, numero maximo de
decrecimiento(-M) 10 y cantidad de procesos creados(-p) 4
